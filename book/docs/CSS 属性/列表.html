list-style

语法：

<ul>
    <li>Coffee</li>
    <li>Tea</li>
    <li>Coca Cola</li>
</ul>

ul {
    list-style: square inside url('arrow.gif');
}

效果：

  • Coffee
  • Tea
  • Coca Cola
