## content

语法：

```html
<a href="https://github.com/" target="_blank" title="GitHub"></a>
<a href="https://www.google.com/" target="_blank" title="Google"></a>
```

```css
a:after {
    content: attr(title);
}
```

效果：

<section id="section_1">
    <a href="https://github.com/" target="_blank" title="GitHub"></a>
    <a href="https://www.google.com/" target="_blank" title="Google"></a>
</section>
