bottom

语法：

<section>
    <p>这是一段普通的段落。</p>
</section>

section {
    position: relative;
    height: 100px;
}

p {
    position: absolute;
    bottom: 10px;
}

效果：

这是一段普通的段落。

clear

语法：

<div>段落一</div>
<p>段落二</p>

div {
    float: left;
}

p {
    clear: both;
}

效果：

段落一

段落二
