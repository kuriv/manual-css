CSS 选择器

CSS 选择器

.class

语法：

<p class="section-class">这是一段普通的段落。</p>

.section-class {
    background-color: yellow;
}

效果：

这是一段普通的段落。

#id

语法：

<p id="section-id">这是一段普通的段落。</p>

#section-id {
    background-color: yellow;
}

效果：

这是一段普通的段落。

*

语法：

<p>这是一段普通的段落。</p>

* {
    background-color: yellow;
}

效果：

这是一段普通的段落。

element

语法：

<div>这是一段普通的段落。</div>
<p>这是一段普通的段落。</p>

p {
    background-color: yellow;
}

效果：

这是一段普通的段落。

这是一段普通的段落。

element, element

语法：

<div>这是一段普通的段落。</div>
<p>这是一段普通的段落。</p>

div, p {
    background-color: yellow;
}

效果：

这是一段普通的段落。

这是一段普通的段落。

element element

语法：

<div>
    <p>
        <span>这是一段普通的段落。</span>
    </p>
</div>

    background-color: yellow;
}

效果：

这是一段普通的段落。

[attribute]

语法：

<a href="https://github.com/" target="_blank">GitHub</a>

a[target] {
    background-color: yellow;
}

效果：

GitHub
