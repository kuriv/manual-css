## list-style

语法：

```html
<ul>
    <li>Coffee</li>
    <li>Tea</li>
    <li>Coca Cola</li>
</ul>
```

```css
ul {
    list-style: square inside url('arrow.gif');
}
```

效果：

<ul style="list-style: square inside url('arrow.gif');">
    <li>Coffee</li>
    <li>Tea</li>
    <li>Coca Cola</li>
</ul>
